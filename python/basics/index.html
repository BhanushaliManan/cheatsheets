  2. Using Variables to store Values
    3. name = "John"
favourite_number = 1
likes_pizza = True
plays_football = False

    In the Above Code, "John" is a String, 1 is an Integer, and The Values True and False are called Booleans,
    And They are stored in the variables "name", "favourite_number", "likes_pizza" and "plays_football" Respectively

  3. Type Conversion

    4. To convert any type into Integers, One can do
    number = "1"
    Here the Number '1' is a String 
    number = int("1")
    But Now it is an Integer
  4. Taking Input from the User
