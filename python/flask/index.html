This Flask Cheatsheet is For You!
  1. Installation 
    pip install flask
  2. Installation of "dotenv" PackageThis Package will be Used for Loading Environment Variables 
    pip install python-dotenv

  3. Basic Directory Structure 
    Flask Project/
└── app.py
└── templates/
└── .env
  4. Boilerplate 
    from flask import Flask
from dotenv import load_dotenv
from pathlib import Path
import os

# Loading Environment Variables
env_path = Path('.') / '.env'
load_dotenv(env_path)

app = Flask(__name__)

# Define a "SECRET_KEY" Variable in your ".env" File
# So your ".env" FIle Should Be Like:

"""
SECRET_KEY=my_super_secret_key
"""

app.config["SECRET_KEY"] = os.getenv("SECRET_KEY")
